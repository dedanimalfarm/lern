# Тестовые базы данных для практики

Для глубокого изучения SQL и администрирования PostgreSQL (особенно тем, связанных с оптимизацией запросов, индексами и резервным копированием) искусственно сгенерированных данных часто бывает недостаточно. 

В этой директории находятся инструменты для загрузки классических тестовых баз данных, которые стали индустриальными стандартами для обучения.

## 1. База "Авиаперевозки" (от Postgres Professional)

Знаменитая демонстрационная база данных, моделирующая систему бронирования авиабилетов по России. Отлично подходит для изучения сложных агрегаций, оконных функций и работы с JSON.

**Как развернуть:**
Запустите скрипт `get_demo_flights.sh`, который скачает архив `demo-small.zip` (содержит данные за 1 месяц, распакованный SQL весит около 60 МБ) с официального образовательного портала PostgresPro и загрузит его в ваш локальный PostgreSQL.
```bash
./get_demo_flights.sh
```
База появится под именем `demo`.

## 2. Pagila (DVD Rental)

Pagila — это официальный порт классической базы данных `Sakila` из MySQL в PostgreSQL. Моделирует бизнес-процессы сети проката DVD-дисков (клиенты, фильмы, актеры, платежи, магазины). Включает в себя использование пользовательских типов данных, триггеров и полнотекстового поиска.

**Как развернуть:**
Запустите скрипт `get_pagila.sh`, который скачает схему и данные из официального репозитория и развернет базу `pagila`.
```bash
./get_pagila.sh
```

## 3. Northwind

Легендарная база данных от Microsoft, портированная на PostgreSQL. Моделирует торговую компанию (заказы, клиенты, сотрудники, товары, поставщики). Отличается очень простой и понятной структурой, идеально подходит для новичков.

**Как развернуть:**
Запустите скрипт `get_northwind.sh`, который скачает и развернет базу `northwind`.
```bash
./get_northwind.sh
```
