# 03 — Каталог статус-кодов своими руками

## Задача
Получить от стенда каждый из кодов: **301, 400, 404, 405, 415, 422** —
и записать в `/tmp/api-lab/m01-codes.txt` по строке на код в формате:

```
404 GET /api/v1/ticket — опечатка в пути
405 DELETE /api/v1/tickets — метод не определён на коллекции
...
```

Какие запросы вызывают какие коды — см. Часть 3 README (но сначала
попробуйте вспомнить без подсказки: это вопрос собеседования).

## Проверка
```bash
for c in 301 400 404 405 415 422; do grep -q "^$c " /tmp/api-lab/m01-codes.txt \
  && echo "$c ok" || echo "$c НЕТ"; done
```

## Ожидаемый результат
Все шесть кодов присутствуют в файле, каждый с пояснением, чем вызван.
