# 03 — Эскалация инцидента в L3

## Задача
Включить `error500`, собрать доказательства (команды из Инцидента 2)
и написать эскалацию в `/tmp/api-lab/m06-escalation.md` строго по шаблону —
пять разделов второго уровня:

```markdown
## Симптом
## Воспроизведение
## Ожидаемо / Факт
## Что уже проверено
## Workaround
```

Требования: точные команды и их фактический вывод; время с таймзоной;
масштаб (какие пути затронуты, какие нет); БЕЗ секретов. После сбора
доказательств поломку выключить (`{"mode":"none"}`).

Эталон для сравнения (после написания своей!):
`solutions/02-escalation-example/escalation.md`.

## Проверка
```bash
grep -c '^## ' /tmp/api-lab/m06-escalation.md   # 5
```

## Ожидаемый результат
Файл с пятью разделами, по которому L3 может начать работу без
дополнительных вопросов.
