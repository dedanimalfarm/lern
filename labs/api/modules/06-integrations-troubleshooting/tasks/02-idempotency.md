# 02 — Победить дубли идемпотентным ключом

## Задача
Отправить **дважды** один и тот же POST с заголовком
`Idempotency-Key` (любое уникальное значение) и title, содержащим
`idem-m06`. Убедиться, что:
- оба ответа — 201 с ОДИНАКОВЫМ id;
- второй ответ помечен `Idempotency-Replayed: true`;
- тикет в системе ровно один.

## Проверка
```bash
curl -s 'http://localhost:8080/api/v1/tickets?q=idem-m06' | jq '.total'   # ровно 1
```

## Ожидаемый результат
`total == 1` при двух отправленных POST.
