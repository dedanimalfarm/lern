# 01 — Обойти пагинацию целиком

## Задача
При `per_page=3` собрать id ВСЕХ тикетов в `/tmp/api-lab/m03-ids.txt`
(по одному id на строку). Количество страниц взять из ответа API,
не хардкодить.

## Подсказка
`pages=$(curl -s '...?per_page=3' | jq .pages)` + цикл `seq 1 $pages`.

## Проверка
```bash
sort -n /tmp/api-lab/m03-ids.txt | head
wc -l < /tmp/api-lab/m03-ids.txt    # == значению .total
```

## Ожидаемый результат
В файле все id (как минимум 1–8 из сид-данных), без пропусков и дублей.
