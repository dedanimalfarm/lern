# 02 — Доказательство 401 vs 403

## Задача
Сохранить в `/tmp/api-lab/m05-403.txt` ПОЛНЫЙ ответ (статус-строка +
заголовки + тело) на попытку агента удалить тикет: DELETE с валидным
токеном агента (флаг `-i` или `-D`).

Рядом (в том же файле или мысленно) — ответ на тот же DELETE вообще без
токена: убедитесь, что это 401, и сформулируйте для себя разницу.

## Проверка
```bash
grep -E '403|forbidden' /tmp/api-lab/m05-403.txt
```

## Ожидаемый результат
Файл содержит `HTTP/1.1 403 Forbidden` и `"code":"forbidden"` с
объяснением про роль agent/admin.
