## Как поднять

```bash
git clone <repo>
cd <repo>
make up            # или другая одна команда
```

Если шаги нельзя свернуть в одну команду — нумерованный список, каждая строка с командой.

## Как проверить, что всё работает

```bash
curl -s http://127.0.0.1:8080/healthz
```

Ожидаемый ответ.

## Как удалить

```bash
docker volume prune -f
rm -rf data/
```
