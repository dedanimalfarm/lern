# Задание 1: Соединение таблиц

База данных: `pagila`

## Шаги выполнения

1. Выведите список всех аренд (id аренды, дата) вместе с именами клиентов (`first_name`), которые их совершили. Используйте `INNER JOIN`.
2. Выведите список абсолютно всех фильмов (`title`) и ID их копий в инвентаре (`inventory_id`). Если фильм отсутствует в инвентаре, вместо ID должен быть `NULL`. Используйте `LEFT JOIN`.
3. Получите полную детализацию аренды №1: выведите имя клиента (`first_name`), дату аренды, название фильма (`film.title`) и сумму платежа (`amount`). Для этого потребуется соединить 4 таблицы: `customer`, `rental`, `inventory` и `film`, а также привязать `payment`. *(В данном задании ограничимся первыми 4 таблицами + payment)*.

## 🏆 Задания для самостоятельной работы

Попробуйте написать SQL-запросы для следующих задач (решения не предоставляются, проверьте себя в консоли psql):

1. **Прокаты конкретного клиента:** Выведите названия всех фильмов, которые когда-либо арендовал клиент с `customer_id = 3`.
2. **Фильтрация соединенных данных:** Выведите список уникальных названий фильмов (`title`) с рейтингом `'R'`, которые были взяты в прокат хотя бы один раз. Используйте `DISTINCT`, чтобы исключить дублирование.
3. **Поиск неиспользуемых адресов:** Напишите Anti-Join запрос, который находит адреса (`address`) из таблицы `address`, на которые не зарегистрирован ни один клиент (отсутствуют в таблице `customer`).
