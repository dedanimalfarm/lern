# Задание 1: Соединение таблиц

База данных: `shop_db`

## Шаги выполнения

1. Выведите список всех заказов (id заказа, дата) вместе с именами пользователей (`name`), которые их совершили. Используйте `INNER JOIN`.
2. Выведите список абсолютно всех пользователей и даты их заказов. Если пользователь не делал заказов, вместо даты должен быть `NULL`. Используйте `LEFT JOIN`.
3. Получите полную детализацию заказа №1: выведите имя пользователя, дату заказа, название товара (`products.name`), количество и цену. Для этого потребуется соединить 4 таблицы: `users`, `orders`, `order_items` и `products`.

## 🏆 Задания для самостоятельной работы

Попробуйте написать SQL-запросы для следующих задач (решения не предоставляются, проверьте себя в консоли psql):

1. **Покупки конкретного клиента:** Выведите названия всех товаров, которые когда-либо заказывал пользователь с `id = 3`.
2. **Фильтрация соединенных данных:** Выведите список всех уникальных заказов (только их `id` и `status`), которые содержат хотя бы один товар из категории `'Электроника'`. Используйте `DISTINCT`, чтобы исключить дублирование заказов в выводе.
3. **Поиск товаров без продаж:** Напишите Anti-Join запрос, который находит названия (`name`) товаров из таблицы `products`, которые ни разу не были куплены (отсутствуют в таблице `order_items`).
