# Задание 1: Соединение таблиц

База данных: `shop_db`

## Шаги выполнения

1. Выведите список всех заказов (id заказа, дата) вместе с именами пользователей (`name`), которые их совершили. Используйте `INNER JOIN`.
2. Выведите список абсолютно всех пользователей и даты их заказов. Если пользователь не делал заказов, вместо даты должен быть `NULL`. Используйте `LEFT JOIN`.
3. Получите полную детализацию заказа №1: выведите имя пользователя, дату заказа, название товара (`products.name`), количество и цену. Для этого потребуется соединить 4 таблицы: `users`, `orders`, `order_items` и `products`.
