# Задание (продвинутое): Соединения (02-joins)

> Это задание повышенной сложности. Базовое задание — в файле `02-joins.md`.

Напишите SQL-запросы в файл `solution.sql`:

1. Разница LEFT и INNER: Выведите названия всех фильмов (`title`) и ID их копий в инвентаре (`inventory_id`). Если фильм отсутствует в инвентаре, он всё равно должен быть в списке (вместо ID копии будет `NULL`).
2. Классический Anti-Join: Найдите названия фильмов, которые **ни разу** не добавлялись в инвентарь (отсутствуют в таблице `inventory`).
3. Соединение множества таблиц: Для аренды с `rental_id = 1` выведите имя клиента (`first_name`), дату аренды (`rental_date`), название фильма (`title`), и сумму платежа (`amount`). Подсказка: соедините `customer`, `rental`, `inventory`, `film` и `payment`.
