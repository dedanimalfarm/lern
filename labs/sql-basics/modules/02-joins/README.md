# Модуль 2: Соединение таблиц (JOIN)

## Теория
Обычно данные разнесены по разным таблицам для избежания дублирования (нормализация). Чтобы получить связанные данные из разных таблиц в одном запросе, используется `JOIN`.
- `INNER JOIN` (или просто `JOIN`) — возвращает строки, для которых есть совпадения в обеих таблицах.
- `LEFT JOIN` — возвращает все строки из левой таблицы и совпавшие из правой. Если совпадений нет, подставляет `NULL`.

Синтаксис:
```sql
SELECT columns 
FROM table1
JOIN table2 ON table1.id = table2.table1_id;
```
