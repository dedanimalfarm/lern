# Задание (продвинутое): Агрегации (03-aggregations)

> Это задание повышенной сложности. Базовое задание — в файле `03-aggregations.md`.

Напишите SQL-запросы в файл `solution.sql`:

1. Подсчитайте общую сумму платежей (`amount`) для каждого клиента (`customer_id`) в таблице `payment`. Выведите `customer_id` и `total_sum`.
2. Фильтрация групп (HAVING): Выведите только те `customer_id`, общая сумма платежей которых превышает 180.00.
3. MIN / MAX: Выведите дату самой первой (`MIN`) и самой последней (`MAX`) аренды в магазине (таблица `rental`, колонка `rental_date`).
4. Агрегация строк: Соберите названия всех фильмов, арендованных клиентом с `customer_id = 1`, в одну строку через запятую. Используйте функцию `string_agg(film.title, ', ')` и соединение с `rental` и `inventory`.
5. Временные ряды (date_trunc): Сгруппируйте аренды по дням (применив функцию `date_trunc('day', rental_date)`). Выведите дату (как `rental_day`) и количество сделанных в этот день аренд (как `rental_count`). Отсортируйте по дате по возрастанию.
