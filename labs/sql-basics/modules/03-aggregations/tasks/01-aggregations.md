# Задание 1: Статистика магазина

База данных: `shop_db`

## Шаги выполнения

1. Подсчитайте общее количество заказов в системе (таблица `orders`).
2. Найдите среднюю цену товаров в каждой категории (таблица `products`). Используйте `GROUP BY`.
3. Подсчитайте, сколько заказов сделал каждый пользователь (нужно соединить `users` и `orders` и сгруппировать по имени).
4. Выведите имена только тех пользователей, которые сделали **более одного** заказа (используйте `HAVING`).
