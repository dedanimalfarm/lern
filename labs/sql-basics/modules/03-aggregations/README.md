## Часть 1: Агрегатные функции
```sql
SELECT COUNT(*), AVG(price) FROM products;
```

---

## Часть 2: GROUP BY и HAVING
```sql
SELECT category, AVG(price) FROM products GROUP BY category HAVING AVG(price) > 10000;
```
