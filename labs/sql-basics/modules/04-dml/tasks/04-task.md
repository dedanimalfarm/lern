# Задание (продвинутое): DML (04-dml)

> Это задание повышенной сложности. Базовое задание — в файле `04-dml.md`.

Напишите SQL-запросы в файл `solution.sql`:

1. Опасность UPDATE: Оберните команду понижения стоимости аренды на 50% (`rental_rate = rental_rate * 0.5`) для всех фильмов в транзакцию с откатом (`BEGIN; ... ROLLBACK;`).
2. Фишка RETURNING: Добавьте нового актера с именем "TEST" и фамилией "ACTOR" в таблицу `actor` и сразу верните его сгенерированный `actor_id`.
3. UPSERT: У нас есть таблица `actor`. Выполните вставку нового актера (`actor_id = 9999`, `first_name = 'TEST'`, `last_name = 'ACTOR'`). Если актер с таким `actor_id` уже существует, обновите его время последнего изменения (`last_update = NOW()`). Используйте `ON CONFLICT (actor_id) DO UPDATE SET ...`.
