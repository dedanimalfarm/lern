# Задание 1: CTE и простые Представления (Views)

База данных: `shop_db`

## Шаги выполнения

Вам необходимо написать SQL-запросы для решения аналитических задач с использованием временных таблиц (CTE) и созданием стандартных представлений (VIEW).

Напишите SQL-запросы для следующих действий:

1. **Использование CTE для расчета среднего чека:**
   Напишите запрос с использованием конструкции `WITH`, который:
   * В CTE (назовите его `order_sums`) считает общую сумму для каждого заказа в таблице `order_items` (сумма рассчитывается как `quantity * price_per_unit`, сгруппированная по `order_id`).
   * В основном `SELECT` рассчитывает среднее значение (`AVG`) полученных сумм заказов из CTE `order_sums`.

2. **Создание стандартного представления (VIEW):**
   Создайте обычное представление с именем `active_products`, которое возвращает все товары из таблицы `products`, цена (`price`) которых строго больше `1000`.
   *(Подсказка: используйте `CREATE VIEW ... AS SELECT ...`).*

3. **Проверка работы представления:**
   Выполните простой `SELECT * FROM active_products;`, чтобы убедиться, что представление возвращает только дорогие товары.

4. **Очистка стенда:**
   Удалите созданное представление `active_products`, чтобы вернуть базу к исходному состоянию.
   *(Подсказка: используйте `DROP VIEW ...`).*
