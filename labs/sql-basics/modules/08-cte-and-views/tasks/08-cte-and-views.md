# Задание 1: CTE и простые Представления (Views)

База данных: `pagila`

## Шаги выполнения

Вам необходимо написать SQL-запросы для решения аналитических задач с использованием временных таблиц (CTE) и созданием стандартных представлений (VIEW).

Напишите SQL-запросы для следующих действий:

1. **Использование CTE для расчета среднего платежа:**
   Напишите запрос с использованием конструкции `WITH`, который:
   * В CTE (назовите его `customer_payments`) считает общую сумму платежей (`amount`) для каждого клиента в таблице `payment` (группировка по `customer_id`).
   * В основном `SELECT` рассчитывает среднее значение (`AVG`) полученных сумм платежей из CTE `customer_payments`.

2. **Создание стандартного представления (VIEW):**
   Создайте обычное представление с именем `premium_films`, которое возвращает все фильмы из таблицы `film`, стоимость аренды (`rental_rate`) которых строго больше `4.00`.
   *(Подсказка: используйте `CREATE VIEW ... AS SELECT ...`).*

3. **Проверка работы представления:**
   Выполните простой `SELECT * FROM premium_films;`, чтобы убедиться, что представление возвращает только дорогие фильмы.

4. **Очистка стенда:**
   Удалите созданное представление `premium_films`, чтобы вернуть базу к исходному состоянию.
   *(Подсказка: используйте `DROP VIEW ...`).*
