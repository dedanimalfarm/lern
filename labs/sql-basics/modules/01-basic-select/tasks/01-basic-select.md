# Задание 1: Выборка и фильтрация

База данных: `pagila`

## Шаги выполнения

1. Выведите названия (`title`) и арендные ставки (`rental_rate`) всех фильмов из таблицы `film`.
2. Выведите все данные о фильмах с рейтингом 'G', арендная ставка которых меньше 3.00.
3. Выведите топ-3 самых длинных фильмов (отсортировать по `length` по убыванию и ограничить вывод).
4. Выведите ID аренды (`rental_id`) из таблицы `rental`, у которых не заполнена дата возврата (поле `return_date` равно `NULL`).

3. **Поиск заполненных данных:** Выведите ID аренды (`rental_id`) из таблицы `rental`, у которых фильмы уже возвращены (поле `return_date` не равно `NULL`).
