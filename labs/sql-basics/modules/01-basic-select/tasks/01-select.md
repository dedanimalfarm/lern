# Задание 1: Выборка и фильтрация

База данных: `shop_db`

## Шаги выполнения

1. Выведите имена (`name`) и цены (`price`) всех товаров из таблицы `products`.
2. Выведите все данные о товарах из категории 'Электроника', цена которых меньше 100000.
3. Выведите топ-3 самых дорогих товаров в магазине (отсортировать по цене по убыванию и ограничить вывод).
4. Выведите имена (`name`) всех пользователей из таблицы `users`, у которых заполнена дата регистрации (поле `registration_date` не равно `NULL`).

---

## 🏆 Задания для самостоятельной работы

Попробуйте написать SQL-запросы для следующих задач (решения не предоставляются, проверьте себя в консоли psql):

1. **Поиск по первой букве:** Выведите всех пользователей из таблицы `users`, чьи имена начинаются на букву `'A'` (или русскую `'А'`).
2. **Множественная сортировка:** Выведите все товары с ценой в диапазоне от `5000` до `30000`. Отсортируйте результат сначала по категории (по возрастанию), а для одинаковых категорий — по цене (по убыванию).
3. **Поиск незаполненных данных:** Выведите имена всех пользователей, у которых не заполнена электронная почта (поле `email` равно `NULL`). *Примечание: на текущих тестовых данных таких пользователей нет, но ваш запрос должен быть синтаксически готов к их появлению.*
