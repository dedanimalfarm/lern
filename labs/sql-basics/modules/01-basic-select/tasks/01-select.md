# Задание 1: Выборка и фильтрация

База данных: `shop_db`

## Шаги выполнения

1. Выведите имена (`name`) и цены (`price`) всех товаров из таблицы `products`.
2. Выведите все данные о товарах из категории 'Электроника', цена которых меньше 100000.
3. Выведите топ-3 самых дорогих товаров в магазине (отсортировать по цене по убыванию и ограничить вывод).
4. Выведите имена (`name`) всех пользователей из таблицы `users`, у которых заполнена дата регистрации (поле `registration_date` не равно `NULL`).
