# Задание (продвинутое): Основы выборки (01-basic-select)

> Это задание повышенной сложности. Базовое задание — в файле `01-basic-select.md`.

Напишите SQL-запросы в файл `solution.sql` в этой же папке:

1. Выведите названия (`title`) и стоимость возмещения (`replacement_cost`) **второй десятки** фильмов с самой высокой стоимостью возмещения. (Сортировка по убыванию стоимости, пропустить первые 10, взять следующие 10).
2. Найдите названия (`title`) и описания (`description`) всех фильмов, в описании которых встречается слово `robot`, без учета регистра символов (например, `Robot` или `ROBOT` также должны попадать в выборку). Подсказка: используйте оператор `ILIKE`.
3. Выведите ID аренды (`rental_id`) и даты аренды (`rental_date`) для всех записей из таблицы `rental`, которые были сделаны в мае (`5`-й месяц) любого года. Подсказка: используйте функцию `EXTRACT(MONTH FROM rental_date)`.
