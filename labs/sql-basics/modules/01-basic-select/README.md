# Модуль 1: Основы выборки данных (SELECT)

## 📖 Теоретическая часть (Что нужно знать)

Перед выполнением практических заданий необходимо самостоятельно изучить следующие темы:
- Базовый синтаксис `SELECT ... FROM ...`.
- Фильтрация строк с помощью `WHERE` и операторы сравнения (`=`, `>`, `<`, `IN`, `BETWEEN`, `LIKE`).
- Логические операторы `AND`, `OR`, `NOT`.
- Сортировка результатов с помощью `ORDER BY` (направления `ASC` и `DESC`).
- Ограничение выборки с помощью `LIMIT` и пагинация с `OFFSET`.
