# Модуль 1: Основы выборки данных (SELECT)

## Теория
Команда `SELECT` используется для извлечения данных из базы.
- `SELECT * FROM table;` — выбрать все столбцы.
- `SELECT col1, col2 FROM table;` — выбрать конкретные столбцы.
- `WHERE condition` — фильтрация строк (операторы `=`, `>`, `<`, `>=`, `<=`, `<>`, `LIKE`, `IN`, `BETWEEN`).
- `ORDER BY col DESC` — сортировка результатов (по убыванию).
- `LIMIT n` — ограничение количества возвращаемых строк.
