# Задание 1: Управление доступом (Roles & Permissions)

База данных: `pagila`

## Шаги выполнения

Вам необходимо создать простого пользователя с ограниченными правами для ручной проверки работы СУБД.

Напишите SQL-запросы для следующих действий:

1. **Создание пользователя:**
   Создайте пользователя (роль с возможностью входа) с именем `test_reader` и паролем `test_pass`.
   *(Подсказка: используйте `CREATE USER ... WITH PASSWORD ...`).*

2. **Выдача прав на подключение:**
   Разрешите пользователю `test_reader` подключаться к базе данных `pagila`.
   *(Подсказка: используйте `GRANT CONNECT ON DATABASE ...`).*

3. **Выдача прав на использование схемы:**
   Разрешите пользователю `test_reader` использовать схему `public`.
   *(Подсказка: используйте `GRANT USAGE ON SCHEMA ...`).*

4. **Выдача прав на чтение конкретной таблицы:**
   Выдайте пользователю `test_reader` права на выполнение `SELECT` только для таблицы `customer`.
   *(Подсказка: используйте `GRANT SELECT ON TABLE ...`).*

5. **Очистка стенда:**
   Удалите созданного пользователя `test_reader`, чтобы вернуть базу в исходное состояние.
   *(Подсказка: используйте `DROP ROLE ...`, но перед этим обязательно отзовите или сбросьте его права с помощью `DROP OWNED BY ...`).*
