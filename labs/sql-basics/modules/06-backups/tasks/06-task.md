# Задание (продвинутое): Резервное копирование и восстановление (06-backups)

> Это задание повышенной сложности. Базовое задание — в файле `06-backups.md`.

Напишите Bash-скрипт в файл `solution.sh` в этой же папке:

1. **Динамический бэкап:**
   Скрипт должен получить текущую дату в формате `YYYY-MM-DD` (например, `2026-06-13`) и сделать резервную копию базы данных `pagila` в формате `Custom` (`-F c`) в файл `/tmp/pagila_YYYY-MM-DD.dump`.
   
2. **Имитация сбоя:**
   Скрипт должен удалить таблицу `film_category` из базы данных `pagila` с помощью консольной команды `psql` (например, через `psql -c "DROP TABLE..."`).

3. **Выборочное восстановление:**
   Скрипт должен восстановить только таблицу `film_category` из только что созданного файла бэкапа обратно в `pagila` с помощью `pg_restore`.

4. **Вывод статуса:**
   В самом конце, при успешном завершении всех шагов, скрипт должен вывести в консоль сообщение:
   `✅ Резервное копирование и выборочное восстановление успешно завершено!`

**Примечания по правам доступа:**
Скрипт будет запускаться от имени суперпользователя. Для работы с СУБД вы можете использовать стандартные команды вызова утилит (например, `pg_dump -d pagila...` или `sudo -u postgres pg_dump -d pagila...`). Убедитесь, что скрипт не требует ручного ввода паролей.
