# Задание (продвинутое): Индексы и оптимизация (05-indexes)

> Это задание повышенной сложности. Базовое задание — в файле `01-indexes.md`.

Напишите SQL-запросы в файл `solution.sql` в этой же папке:

1. **Диагностика сложного поиска:**
   Напишите запрос, который выбирает все товары категории `'Электроника'`, цена которых строго больше `50000`. Оберните его в команду `EXPLAIN ANALYZE`, чтобы посмотреть, как СУБД сканирует таблицу без дополнительных индексов.
   
2. **Создание составного индекса:**
   Напишите команду создания составного индекса (Composite Index) `idx_products_cat_price` для таблицы `products` по полям `category` и `price`.

3. **Проверка составного индекса:**
   Напишите блок команд, который временно отключает `enable_seqscan`, делает `EXPLAIN ANALYZE` для запроса из шага 1 (чтобы убедиться в использовании нового составного индекса), а затем возвращает `enable_seqscan` в исходное состояние (`RESET enable_seqscan;`).

4. **Очистка:**
   Напишите команду удаления индекса `idx_products_cat_price`, чтобы вернуть состояние базы данных к исходному.
