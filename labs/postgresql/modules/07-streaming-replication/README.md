# Модуль 7: Потоковая репликация (Streaming Replication)

> 🚧 **Статус:** В разработке (Трек 2: Production Readiness)

## Планируемые темы
- Понятие Write-Ahead Logging (WAL).
- Настройка Primary сервера.
- Инициализация Replica сервера с помощью `pg_basebackup`.
- Настройка асинхронной потоковой репликации.
- Мониторинг отставания реплики (replication lag).
- (Опционально) Синхронная репликация.
