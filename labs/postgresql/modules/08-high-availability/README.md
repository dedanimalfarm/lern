# Модуль 8: High Availability (Patroni)

> 🚧 **Статус:** В разработке (Трек 2: Production Readiness)

## Планируемые темы
- Почему стандартной репликации недостаточно для отказоустойчивости (ручной failover).
- Введение в Patroni: архитектура и принципы работы.
- Развертывание DCS (Distributed Configuration Store) — etcd или Consul.
- Настройка кластера Patroni.
- Эмуляция падения Primary и проверка автоматического failover.
- Интеграция Patroni с HAProxy или PgBouncer.
