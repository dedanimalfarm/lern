# 02 — Посчитать syscalls по процессам (агрегат в ядре)

## Задача
Использовать eBPF-map для подсчёта системных вызовов по процессам — агрегация
происходит В ЯДРЕ, поэтому оверхед минимален (не передаём каждое событие в userspace).

## Проверка
```bash
sudo bpftrace -e 'tracepoint:raw_syscalls:sys_enter { @sys[comm] = count(); }'
# Ctrl-C через пару секунд — bpftrace распечатает map
```

## Ожидаемый результат
```
@sys[timeout]: 9
@sys[bpftrace]: 22
@sys[postgres]: 239        # самый «болтливый» процесс по числу syscalls
```
`@sys[comm] = count()` копит счётчики в eBPF-map прямо в ядре; по выходу bpftrace
печатает их. `raw_syscalls:sys_enter` ловит ВСЕ вызовы (а не один конкретный) —
удобно искать процесс, который «забивает» ядро.
