# 03 — Читать и декодировать capabilities

## Задача
Научиться смотреть пять наборов привилегий процесса (`/proc/<pid>/status`) и
декодировать hex-маски в человекочитаемые имена.

## Проверка
```bash
grep Cap /proc/self/status         # пять наборов в hex
capsh --print | head -2            # то же человекочитаемо
capsh --decode=0000000000000001    # один бит
capsh --decode=000001ffffffffff    # полный root-набор
```

## Ожидаемый результат
```
CapInh:  0000000000000000
CapPrm:  000001ffffffffff
CapEff:  000001ffffffffff     # root: включены все ~41
CapBnd:  000001ffffffffff
CapAmb:  0000000000000000
...
0x0000000000000001=cap_chown
```
- `CapEff` — что действует сейчас, `CapPrm` — что можно включить, `CapBnd` —
  потолок (нельзя получить выше), `CapAmb` — ambient (текут через execve).
- Декодирование показывает, какие именно биты привилегий выставлены.
